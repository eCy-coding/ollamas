Embeddable chat widget

This page is a plain host site. The streaming chat in the corner is the entire ollamas widget — one zero-dependency script, isolated in a Shadow DOM.

One tag

Add the widget to any website:

<script src="/embed.js"
  data-api-base="https://your-ollamas"
  data-provider="ollama-local"
  data-model="llama3"
  data-title="ollamas"></script>

What it does

Streams replies from POST /api/generate over SSE. No build step, no framework, no host-CSS conflicts.
