# MCP Tool Registry
2
3
## Overview
4
5
The /mcp endpoint is the single choke-point — every tool call routes through one **ToolRegistry**, gated by tier (safe, host, privileged) before it touches the host.
6
7
- Single execute() choke-point — no tool runs outside the registry
8
- autoApply = false by default — a human write-gate on every privileged call
9
- Upstream federation — remote servers mount as namespaced sub-registries
10
11
```ts
12
enum ToolTier {
13
Safe = 0, // read-only, no side effects
14
Host = 1, // touches filesystem or network
15
Privileged = 2, // mutates host state — always gated
16
}
17
```
18
19

20
21
- [ ] Registry rejects calls to unregistered tool ids
22
- [ ] Privileged calls block until a human approves
- [ ] Upload rejects .exe disguised as .pdfTab ⇥ accept
